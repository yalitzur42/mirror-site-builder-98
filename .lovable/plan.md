

# תוכנית: עדכון עיצוב האתר - תמונות, צבעים וגופן

## סקירה כללית
התוכנית כוללת שלושה שינויים מרכזיים:
1. יצירת תמונות AI מקצועיות לכל האתר
2. מעבר לסכמת צבעים דואלית (חום על שמנת / שמנת על חום)
3. שינוי הגופן לגופן אלגנטי יותר

---

## שלב 1: שינוי הגופן

**גופן חדש**: Rubik (גופן עברי מודרני ואלגנטי)
- מתאים לעברית RTL
- נקי ומקצועי
- תומך במשקלים שונים (300-900)

**קבצים לעדכון**:
- `src/index.css` - החלפת ייבוא Google Fonts
- `tailwind.config.ts` - עדכון fontFamily

---

## שלב 2: סכמת צבעים דואלית

**הצבעים**:
- שמנת: `#F5F5DC` (HSL: 60 56% 91%)
- חום: `#4B2E1A` (HSL: 25 50% 20%)

**מבנה הסקציות בדף הבית**:
```text
+---------------------------+
|  Hero (חום + שמנת)         |
+---------------------------+
|  שירותים (שמנת + חום)       |
+---------------------------+
|  אקדמיה (חום + שמנת)        |
+---------------------------+
|  גלריה (שמנת + חום)         |
+---------------------------+
|  פרם (חום + שמנת)           |
+---------------------------+
|  מוצרים (שמנת + חום)        |
+---------------------------+
|  אודות (חום + שמנת)         |
+---------------------------+
|  צוות (שמנת + חום)          |
+---------------------------+
|  CTA (חום + שמנת)           |
+---------------------------+
```

**שינויים בקוד**:
- הוספת CSS variables חדשים לסקציות בהירות
- הוספת utility classes: `.section-light` ו-`.section-dark`
- עדכון כל הדפים לשימוש בסקציות לסירוגין

---

## שלב 3: יצירת תמונות AI

**רשימת תמונות ליצירה** (כ-20 תמונות):

### דף הבית:
1. **Hero** - ספר גברים מקצועי עובד על לקוח, סגנון גברי ואלגנטי
2. **גלריה** (8 תמונות) - תספורות גברים שונות, פרמים, עיצוב זקן
3. **פרם** - גבר עם תלתלים מושלמים
4. **מוצרים** (4 תמונות) - שמן זקן, ווקס, שמפו, סרום
5. **SEO** - פנים מספרה מודרנית
6. **צוות** - דיוקן ספר מקצועי

### דפים נוספים:
7. **About Hero** - צוות מספרה
8. **Barbershop Hero** - תמונת מספרה
9. **Perm Hero** - גבר עם תלתלים
10. **Academy Hero** - סביבת לימודים
11. **Shop** (8 תמונות) - מוצרי טיפוח

## פרטים טכניים

### CSS Variables החדשים:
```css
:root {
  /* Base (Dark/Brown) */
  --background: 25 50% 20%;
  --foreground: 60 56% 91%;
  
  /* Light Section (Cream) */
  --light-bg: 60 56% 91%;
  --light-fg: 25 50% 20%;
}
```

### Utility Classes:
```css
.section-dark {
  @apply bg-background text-foreground;
}

.section-light {
  background-color: #F5F5DC;
  color: #4B2E1A;
}
```

### יצירת תמונות:
- שימוש ב-Nano banana model
- סגנון: צילום מקצועי, תאורה חמה, אווירה גברית
- שמירה בתיקיית `src/assets/`
